# Website - Dra. Diana Cruz Martins

Landing page profissional em português de Portugal para a Dra. Diana Cruz Martins.

## Como editar

Os textos, links, publicações, FAQ, áreas clínicas e dados institucionais estão centralizados em:

`src/data/profile.js`

Os campos marcados com `VALIDAR` devem ser confirmados pela médica antes de publicar, sobretudo formação académica, publicações finais, fotografia oficial, respostas clínicas de FAQ e links profissionais.

## Fotografia

Adicionar a fotografia oficial autorizada em:

`images/diana-cruz-martins-cuf.jpg`

Retrato complementar usado na secção "Sobre":

`images/diana-cruz-martins-retrato.jpg`

## Pré-visualização

Por usar módulos JavaScript simples, a página deve ser vista através de um servidor local. Um servidor estático é suficiente.
